# Product Overview

This is an admin dashboard application built as a monorepo project. The application provides a web-based administrative interface with routing, drag-and-drop functionality, and a sidebar-based navigation system.

## Key Features

- Admin dashboard with sidebar navigation
- Multiple routes and pages for different admin functions
- Drag-and-drop tools using dnd-kit
- Responsive layout with header and main content areas
- Component-based UI using Radix UI primitives
